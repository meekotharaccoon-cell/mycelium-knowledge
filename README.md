## What this is

Mycelium Knowledge is the intelligence infrastructure of SolarPunk. Every cycle, the organism:

- Harvests content from HackerNews, Reddit, DEV.to, and RSS feeds
- Watches AI model releases and open-source tool launches
- Synthesizes insights into product ideas and content angles
- Stores everything in a structured memory that persists across cycles
- Uses that memory to make better decisions next cycle

This is how SolarPunk gets smarter over time without human input.
